# Quantum Cloud

Premium futuristic website for Quantum Cloud (QC), a private software development and advanced technology company.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

## Project Notes

- Built with Next.js App Router, Tailwind CSS, Framer Motion, and GSAP.
- Uses a system-first luxury font stack to ensure reliable rendering in offline environments.
